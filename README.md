# rollup
Rollup 是一个 JavaScript 模块打包器，可以将小块代码编译成大块复杂的代码，例如 library 或应用程序。
## rollup-basic 

基于rollup官方文档的demo
+ 添加了json处理插件可以读取json文件的内容
+ 添加了rollup配置文件
+ 添加了`npm run build`命令,可以通过该命令进行打包

> [rollupjs-官方文档-教程](https://www.rollupjs.com/guide/tutorial)

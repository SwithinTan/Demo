# 简介
本项目为rollup demo

# 项目结构

## rollup-basic

基于写的rollup基础例子
> [Rollup 打包并发布到 npm](https://blog.csdn.net/tianxintiandisheng/article/details/118369404)

## rollup-npm-package

rollup引入npm依赖包
这个 rollup-plugin-node-resolve 插件可以告诉 Rollup 如何查找外部模块
```
npm install --save-dev rollup-plugin-node-resolve
```
